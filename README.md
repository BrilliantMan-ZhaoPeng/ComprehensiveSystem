# ComprehensiveSystem
一个简单综合性的小项目(springboot应用开发流程打通，shiro权限分配)
## 项目介绍
该项目是以springboot为核心进行简单的开发整理，项目中打通前后端的开发，采用**shiro**进行权限的分配，以session进行请求单次提交的设计
